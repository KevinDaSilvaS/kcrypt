# kcrypt
<h3>Kcrypt or Krypt is an encrypting and decrypting lib created using javascript and node js</h3>
<h5>To encrypt:</h5>
<p>//call krypt method</p>
<p>krypt("ARca")</p>
```
//outputs: GIJ<-->XZ[<-->ikl<-->gij<-->
```
 
 <h5>To decrypt:</h5>
<p>//call unkrypt method</p>
```
unkrypt("GIJ<-->XZ[<-->ikl<-->gij<-->")
```
``` 
returns "ARca" 
```
